安装Robotgo

利用go 来操作windows 的键盘鼠标

项目地址

https://github.com/go-vgo/robotgo

安装 go for windows

https://golang.org/dl/

安装git

https://git-scm.com/download/win 记得把git 添加到path 里，即在cmd 里能直接运行git命令

安装 MinGW-w64 需要gcc 这类工具

http://mingw-w64.org/doku.php/download/mingw-builds 国内网络可能有点慢，可以在国外下载好，在打包拷贝回来

运行 mingw-w64.bat

编辑一下mingw-w64.bat里面的路径，如果不对记得改下。 运行后要能运行 git， gcc， go 三个程序， 如果不对前面几步检查一下
